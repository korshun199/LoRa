# LoRa

Универсальная система управления оборудованием через LoRa/Wi-Fi.

## Цель

Создать расширяемую систему, где устройство можно просто включить, подключиться к его Wi-Fi, ввести настройки, а дальше управлять оборудованием через серверное ПО.

## Основной стек

- ESP32 / XIAO ESP32S3
- SX1262 LoRa
- Python FastAPI
- MariaDB
- Android
- Linux / Lubuntu

## Структура

- firmware/ — прошивки устройств
- gateway/ — LoRa/Wi-Fi шлюз
- server/ — серверное ПО
- web/ — веб-интерфейс
- hardware/ — схемы, распиновки, фото
- docs/ — документация
- configs/ — конфиги
- tools/ — утилиты
- tests/ — тесты
- scripts/ — скрипты
