# LoRa Dev Server MVP-0

## Статус

Локальный центральный сервер разработки на ноутбуке работает.

## Назначение

Ноутбук используется как dev-среда.  
VPS позже будет получать код через Git/GitHub и запускать боевую копию серверного ПО.

## Уже работает

- FastAPI server
- virtualenv
- запуск через scripts/run_server_dev.sh
- проверка /health
- регистрация gateway-001
- создание команды для node-001
- чтение команды
- подтверждение выполнения через /command/ack
- просмотр состояния через /api/debug/state
- тест полного цикла через scripts/test_api_cycle.sh

## Проверенный цикл

server -> command -> node -> ack

## Пока временно

Данные хранятся в памяти Python-процесса.  
После перезапуска сервера команды, статусы и ACK очищаются.

## Следующий этап

Подключение MariaDB и сохранение devices, commands, statuses, acks в базу.
