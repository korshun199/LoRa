# LoRa Link Test Web

Тестовая система проверки качества LoRa-связи через Android/Web.

## Идея

Каждое устройство XIAO ESP32S3 + Wio SX1262 прошивается одинаковой прошивкой.

При настройке устройство получает:

- позывной
- IP адрес из сети 192.168.4.0/24

Устройство поднимает Wi-Fi точку доступа и веб-интерфейс.

Android подключается к выбранной точке и через браузер видит соседние LoRa-устройства:

- позывной
- IP
- RSSI
- SNR
- качество связи
- время последнего сигнала

## Первая цель

Сначала сделать простую веб-морду с API:

- GET /api/status
- GET /api/peers
